# 📌 Task Manager CLI (Node.js)

## 📖 Overview
This is a Command-Line Task Management System built using Node.js.

The application allows users to manage tasks directly from the terminal without using any external libraries or frameworks.

---

## 🚀 Features

- ✅ Add Task (Title, Description, Priority, Due Date)
- ✅ Prevent Duplicate Tasks
- ✅ View Tasks (Grouped by Priority)
- ✅ Search Tasks (Title, Status, Priority)
- ✅ Update Task Status
- ✅ Delete Task (with confirmation)
- ✅ Auto Save to JSON File
- ✅ Load Tasks on Startup

---

## 🛠️ Technologies Used

- Node.js
- JavaScript
- Built-in modules:
  - fs
  - readline
  - path
